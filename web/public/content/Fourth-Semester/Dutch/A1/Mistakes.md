## Pronouns

- **Hij/zij/het**:
    - <span style="color:#00bcd4">hem</span> (him) / <span style="color:#e91e63">haar</span> (her) / <span style="color:#9c27b0">het</span> (it)
    - <span style="color:#00bcd4">zijn</span> (his) / <span style="color:#e91e63">haar</span> (her) / <span style="color:#9c27b0">zijn</span> (its) <br> <span style="font-size:0.9em;color:gray">*Also for zij/ze*</span>
    - <span style="color:#4caf50">zich</span> (himself/herself/itself) + <span style="color:#ff9800">zelf</span>

## De / Het Articles

| Article | When to Use | Examples |
|:-------:|:-----------:|:---------|
| <span style="color:#03a9f4;font-weight:bold">het</span> | <ul><li>Words starting with <b>be-</b>, <b>ge-</b>, <b>ver-</b>, <b>ont-</b></li><li>Ending in <b>-st</b></li><li>Categories: <span style="color:purple">Languages, countries, places, metal, directions, sports, games</span></li></ul> | <span style="color:#03a9f4">het appartement</span>, <span style="color:#03a9f4">het huis</span>, <span style="color:#03a9f4">het gezin</span>, <span style="color:#03a9f4">het probleem</span> |
| <span style="color:#ff9800;font-weight:bold">de</span> | <ul><li>Categories: <span style="color:purple">Clothings, people, plurals, -ing, -e, -er, -el, -em, -en, -je</span></li></ul> | <span style="color:#ff9800">de familie</span>, <span style="color:#ff9800">de woonruimte</span>, <span style="color:#ff9800">de pagina</span> |

- het gezin (family)
- het pak (suit)
- het einde (end)
- het bureau (desk)
- het voordeel (advantage)
- het probleem (problem)
