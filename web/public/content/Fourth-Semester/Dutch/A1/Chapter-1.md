## Common “See you...” Phrases  
| Dutch              | English                     | Note                   |
|:-------------------|:----------------------------|:-----------------------|
| **Tot zo**         | see you soon                | < 1 hour               |
| **Tot ziens**      | see you                     | formal / open ended    |
| **Tot straks**     | see you soon                | e.g. in 2 hours        |
| **Tot morgen**     | see you tomorrow            |                        |
| **Tot later**      | see you later               | same day               |
